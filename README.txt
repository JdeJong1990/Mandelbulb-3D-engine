This project allows a user to create an image of the Mandelbulb fractal. It consists of the folloing files:
mandelskeleton_03.py: 	This script is for the user to make an image
mandelclass_v04.py: 	This is a class definition for a Mandelbulb object, used to create an image
Jtoolbox.py: 		This is a collection of functions that are used by the mandelclass
